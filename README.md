# TodoListForMyPc
todo 목록을 관리하는 위젯을 못찾아서 직접 만드는 할일 관리 위젯

![1](https://github.com/user-attachments/assets/e25af450-f938-45dc-8973-05a10d5f9d9d)

## 기능
- 할일 생성, 수정, 삭제
    - 생성 시 우선순위 설정 가능
    - 수정 시 Dday 설정 가능
    - 우선순위 수정 가능
    - 우선순위 높아지면 상단 배치
- Dday 다가오면 알림 전송(테스트 미완)
