# TodoListForMyPc
할 일 목록을 관리하는 위젯을 못 찾아서 직접 만드는 할 일 관리 위젯

## 기능
- 할 일 생성, 수정, 삭제
    - 생성 시 우선순위 설정 가능
    - 수정 시 Dday 설정 가능
    - 우선순위 수정 가능
    - 우선순위 높아지면 상단 배치
- Dday 다가오면 알림 전송(테스트 미완)
